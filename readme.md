# Paul Graham Corpus

## About
The collection of Paul Graham's written wisdom. Extracted from http://www.paulgraham.com/articles.html

All individual essays are found on the _txt_ folder as .txt files

The _full_df.csv_ file contains a data frame with metadata for each essay:  
1. title
2. partial_link
3. full_link
4. outlinks
5. word_count
